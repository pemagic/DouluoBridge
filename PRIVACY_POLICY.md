# 隐私政策 / Privacy Policy

**最后更新 / Last Updated: 2026-02-16**

## 中文版

《斗罗大桥：像素剑影》（以下简称"本游戏"）尊重并保护用户的隐私。本隐私政策旨在说明本游戏对用户信息的处理方式。

### 信息收集

**本游戏不会收集、存储、使用或分享任何个人信息。**

具体而言，本游戏：

- ❌ 不收集用户的姓名、邮箱、电话或任何身份信息
- ❌ 不收集设备识别信息（如 IDFA、IDFV）
- ❌ 不收集位置信息
- ❌ 不收集使用行为数据或分析数据
- ❌ 不使用任何第三方分析或广告 SDK
- ❌ 不包含任何广告
- ❌ 不进行任何网络请求或数据传输
- ❌ 不使用 Cookie 或类似追踪技术

### 数据存储

本游戏的所有数据（包括游戏进度）仅存储在用户设备本地，不会上传至任何服务器。

### 第三方服务

本游戏不集成任何第三方服务、SDK 或 API。

### 儿童隐私

本游戏不收集任何用户信息，因此不存在儿童隐私问题。本游戏适合所有年龄段的用户。

### 政策变更

如本隐私政策发生变更，我们将在此页面上更新。

### 联系方式

如有关于本隐私政策的问题，请通过 GitHub Issues 联系我们：
https://github.com/pemagic/DouluoBridge/issues

## English Version

**Douluo Bridge: Pixel Sword** ("the Game") respects and protects user privacy. This Privacy Policy explains how the Game handles user information.

### Information Collection

**This Game does not collect, store, use, or share any personal information.**

Specifically, this Game:

- ❌ Does not collect names, emails, phone numbers, or any identifying information
- ❌ Does not collect device identifiers (such as IDFA or IDFV)
- ❌ Does not collect location data
- ❌ Does not collect usage or analytics data
- ❌ Does not use any third-party analytics or advertising SDKs
- ❌ Does not contain any advertisements
- ❌ Does not make any network requests or data transmissions
- ❌ Does not use cookies or similar tracking technologies

### Data Storage

All game data (including game progress) is stored locally on the user's device and is never uploaded to any server.

### Third-Party Services

This Game does not integrate any third-party services, SDKs, or APIs.

### Children's Privacy

Since this Game does not collect any user information, there are no children's privacy concerns. This Game is suitable for users of all ages.

### Policy Changes

If this Privacy Policy is updated, changes will be posted on this page.
