# Douluo Bridge Audio Assets

To enable real audio, please place the following MP3 files in this directory:

## Background Music (BGM)
- `bgm_0.mp3` : Level 1 (Cang Hai Yi Sheng Xiao / 沧海一声笑)
- `bgm_1.mp3` : Level 2 (Purple Bamboo Melody / 紫竹调)
- `bgm_2.mp3` : Level 3 (Dance of Golden Snake / 金蛇狂舞)
- `bgm_3.mp3` : Level 4 (Ambush from All Sides / 十面埋伏)
- `bgm_4.mp3` : Level 5 (Moon Over Mountain Pass / 关山月)
- `bgm_5.mp3` : Level 6 (General's Command / 将军令)
- `bgm_6.mp3` : Level 7 (Guangling San / 广陵散)
- `bgm_7.mp3` : Level 8 (Moon Reflected in Second Spring / 二泉映月)
- `bgm_8.mp3` : Level 9 (Autumn Moon over Han Palace / 汉宫秋月)
- `bgm_9.mp3` : Level 10 (Three Variations of Plum Blossom / 梅花三弄)

## Sound Effects (SFX)
- `shoot.mp3` : Player shooting sound (Retro laser or arrow)
- `slice.mp3` : Dash/Kill skill sound (Watermelon cut / sharp slice)

> **Note**: The game has a fallback system. If these files are missing, it will automatically generate improved synthetic versions.
